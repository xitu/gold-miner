# 掘金翻译计划

翻译优质的互联网技术文章

# 如何参与

1. 从下面的文章列表中挑选出待翻译的文章
2. 在 [issue](https://github.com/xitu/gold-miner/issues/new) 中提交翻译申请
3. 申请请求通过后，Fork 此仓库开始翻译
4. 翻译完成，发送 Pull Request
